# Power BI Chinook Project

A interactive Power BI dashboard built on the **Chinook Sample Database** - a relational database containing music store data (Artists, Albums, Tracks, Customers, Invoices, etc.).

### Key Features
- Sales performance analysis by country, customer, and genre
- Revenue trends and top-performing artists/albums
- Customer segmentation and purchasing behavior
- Interactive visualizations and dynamic filters

### Technologies Used
- Power BI Desktop
- Chinook SQLite Database
- DAX (Data Analysis Expressions)

---
**Dashboard Purpose**: To provide actionable business insights for a digital music store using data visualization and analytics.
